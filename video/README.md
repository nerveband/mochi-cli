# Mochi CLI - Promotional Video

This directory contains a Remotion project for creating the mochi-cli promotional video.

## Brand Colors

- **Background**: `#f7f7f7` (Light gray/white)
- **Text**: `#333333` (Dark gray)
- **Accent Primary**: `#a7ce60` (Lime green)
- **Accent Secondary**: `#b8da79` (Light green)

## Scenes

1. **Intro** (0-2s): Logo reveal with tagline
2. **CLI Demo** (2-7s): Terminal animation showing commands
3. **Features** (7-12s): Grid of 6 key features
4. **Outro** (12-15s): Call-to-action with install command

## Usage

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Build video
npm run build
```

## Output

The rendered video will be saved to `out/demo.mp4`.
